# Demo Flow

Use this short flow for a README GIF, a GitHub release video, or an OSS support application.

## Setup

1. Start the Mac bridge:

   ```sh
   cd mac
   ./start-bridge.sh
   ```

2. Copy the printed Mac URL and token.
3. Install the Android APK.
4. Enter the URL and token in the Android app.
5. Grant overlay permission.

## Live Demo

1. Open Parsec on Android and connect to the Mac.
2. Open any text field on the Mac, such as Notes, a browser search box, or a prompt input.
3. Tap the Parsec Bubble Input floating bubble.
4. Type text or use Android voice input.
5. Tap send.
6. The text appears in the focused Mac app.

## Demo Script

> Parsec gives me the screen and basic control. The awkward part is text input from a phone. Parsec Bubble Input adds a small Android overlay. I type or dictate on Android, tap send, and the Mac bridge pastes the text into the focused window. It is local-first, token-protected, and small enough to self-host.
