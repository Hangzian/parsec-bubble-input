# Changelog

## 1.0.0 - 2026-06-01

Initial open-source release.

- Android floating bubble input companion for Parsec
- Compact overlay input panel
- Send current text to Mac
- Send clipboard text
- Resend last sent text
- Auto-send toggle
- Bubble position persistence
- Token-based Mac bridge
- Optional Python relay for off-LAN use
- MIT license and public setup documentation
